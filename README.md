# NukeJS
What is NukeJS I hear? Its a Framework built for discord.js which allows adds in some extra features.

## Features
- Fallback System (WIP)
- Command Handler
- Ready Message
- Error Logging (including Bot Crash emailing) (WIP)
And much more to come
